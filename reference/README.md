# Reference Subtrees

This repository is now **Elixir-first**.

The top-level `reference/` directory keeps the earlier companion implementations that still provide historical context, snapshot data, and low-level runtime comparison points:

- `reference/python/` — the original clean-room Python mirror subtree
- `reference/rust/` — the Rust runtime/reference subtree

These trees are retained for reference and comparison. The primary developer workflow lives under `elixir/`.
